sshのフォルダに入るコマンド
`cd ~/.ssh`
無ければ
` mkdir ~/.ssh`
`cd ~/.ssh`

鍵作成のコマンド
`ssh-keygen -t rsa`

パスワードを設定しない場合は全てenter

ファイル名の確認コマンド
`ls`

id_rsa　id_rsa.pub が確認できたら完了

文字ファイルの中身を表示
`cat id_rsa.pub`

出てきたkeyをGithubの画面に貼り付け

接続確認用のコマンド
`ssh -T git@github.com`

下記の内容が表示されれば接続完了
Hi {アカウント名}! You've successfully authenticated, but GitHub does not provide shell access.
